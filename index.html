Mr. Roboger's Neighborhood

Created By: Janet Karpenske

This webpage is very special because it allows the user to submit a number and have a range of numbers from 0 up to the inputed number displayed on the screen. The catch is that any number with a 3 in it will say "Won't you be my neighbor?", any number with a 2 in it will say "Boop!", and any number with a 1 in it will say "Beep!"
